\BugKiller\DataBundle\Controller

\BugKiller\DataBundle\ControllerDataController

DataController Definition de la classe DataController utilisée pour toutes les donnée Ce controller gère les verbes http GET,POST,PUT,DELETE à la façon d'un API REST pour toutes les entités de façon générique.

Il implémente la gestion des filtres , des trie et de la pagination en provencance d'ExtJs
